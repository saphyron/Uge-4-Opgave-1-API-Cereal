Detailed Description

Simpelt “factory”-objekt som ud fra en connection string kan udlevere nye databaseforbindelser til SQL Server.

Indkapsler SqlConnection bag IDbConnection for at gøre afhængigheder nemmere at teste og udskifte. Hver kald til Create returnerer en ny, endnu ikke åbnet forbindelse, som kalderen selv skal åbne og dispose via using.
