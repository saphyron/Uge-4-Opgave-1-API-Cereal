CerealAPI.Utils Namespace Reference

Classes

class  CsvParser
 Simpel CSV-parser til Cereal-data: læser semikolon-separerede filer og mapper til Cereal. Spring­er 2. linje (datatype-linje) over og tolker "-1" som null for talfelter.
class  FilterParser
 Parser querystring-filtre til en sikker SQL WHERE-klausul + Dapper-parameterbindinger. Formålet er at understøtte både “rå” udtryk (fx calories>=100) og alias‐nøgler (fx calories_gte=100) uden at åbne for SQL injection.
class  HttpHelpers
 Hjælpefunktioner til at bygge URL-stier og querystrings, samt typed HTTP-kald til Cereal-endpoints. Samler gentagne URL-operationer ét sted.
class  SortParser
 Parser for ?sort=-queryparametret, der konverterer en kommasepareret liste som fx calories_desc,name_asc til et sikkert SQL ORDER BY-fragment og en struktureret liste over sorteringsdele.
