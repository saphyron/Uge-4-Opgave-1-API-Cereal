Detailed Description

Domænemodel for en bruger, som afspejler rækken i dbo.Users. Bruges af Dapper til materialisering af database-rows.

PasswordHash indeholder en hash (ikke klartekst) og må aldrig logges.

Role default'er til "user" og kan opdateres af admins.

CreatedAt sættes i databasen (f.eks. via DEFAULT/GETUTCDATE()).
