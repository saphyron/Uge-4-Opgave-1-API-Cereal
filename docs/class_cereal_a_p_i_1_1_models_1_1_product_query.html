Detailed Description

Query-DTO der binder automatisk fra querystring (via [AsParameters]) og bruges til filtrering/søgning på /products-endpoints.

  • Null-værdier ignoreres; kun udfyldte felter bliver til filtre.
  • nameLike er til delvise match (typisk SQL LIKE).
  • manufacturer er et alias for mfr for mere naturlig klient-API.
  • Numeriske felter er nullable for at undgå 0 som implicit filter.
  • Kan kombineres med endpoint-specifik sortering/paging.
