Detailed Description

Logger hver HTTP-request med metode, sti, statuskode og varighed. Hjælper med at få overblik over trafikken og fejl i applikationen.

Måler varighed med en Stopwatch, lader requesten gå videre i pipeline og logger et Information-entry ved succes samt et Error-entry ved exceptions. Middleware genkaster exception efter logging, så korrekt fejlhåndtering/HTTP-svar bevares længere oppe i pipeline.
